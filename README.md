# coding_collective_test

## Setup
1. Clone the repo
2. open server folder
3. run `composer install`
4. run `php artisan migrate --seed`
5. run `php artisan serve`
6. back to root folder
7. open client folder
8. run `npm install`
9. run `npm run dev`

## User:
`localhost:5173/`

## Admin:
`localhost:5173/admin/dashboard`
